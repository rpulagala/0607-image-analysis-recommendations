🏗️ How It Works

The user uploads an image through the Next.js frontend. The FastAPI backend saves it to Supabase Storage, sends it to GPT-4o Vision for analysis, feeds the structured output into a second AI call to generate personalized recommendations, then writes the result to the database and returns it to the user in real time.

Supabase handles authentication, the database, and file storage in a single platform — this keeps the architecture simple and the cost low at MVP scale. GPT-4o Vision performs image analysis and recommendation generation in one model, reducing latency and complexity.
